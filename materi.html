Roadmap CSS

Ikuti urutan belajar dari CSS dasar sampai styling project kecil

Materi dibagi menjadi fase kecil. Mulai dari menghubungkan CSS, lalu pelan-pelan masuk ke spacing, layout, responsive, dan debugging.

Cara pakai roadmap

Belajar 15-25 menit per sesi

Untuk pemula CSS, fokuslah pada satu perubahan visual dulu. Lihat hasilnya sebelum pindah ke property lain.

  1. Baca masalah dan analogi terlebih dahulu.
  2. Salin contoh HTML + CSS ke editor, lalu ubah satu property.
  3. Bandingkan preview sebelum dan sesudah perubahan.
  4. Tandai selesai jika sudah bisa menjelaskan dengan bahasa sendiri.
