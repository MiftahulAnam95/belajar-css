# Belajar CSS

Website belajar CSS interaktif untuk pemula yang baru selesai belajar HTML dasar dan ingin melanjutkan ke styling halaman.

## Fitur

- 20 materi CSS bertahap setelah HTML dasar
- Analogi, istilah penting, kesalahan umum, dan cek pemahaman
- Quiz, recall challenge, latihan debugging, dan editor live preview
- Ide mini project CSS
- Progress, badge, dan tema gelap yang tersimpan di browser

## Menjalankan

Buka `index.html` langsung di browser, atau jalankan server statis sederhana dari folder ini.
